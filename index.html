HTML Terminology

Forms - Elements

Tag Name Description
<form> Form Defines a form to collect user input
<input> Input Displays depending on the type attribute
<label> Label Defines a label for form elements
<select> Select Defines a drop-down list
<textarea> Text Area Defines a multi-line input field
<datalist> Data List Specifies a list of pre-defined options for an <input>

(Some)Input Types

Type= Use
text Single-line text input field
password Password field. Hides inputted text with dots
reset Defines a button for resetting all form values to default
radio Allows the user to select one of the given choices
checkbox Allows the user to select zero or multiple of the given choices
button Defines a button
range Defines a slider control
date Date field
email Email field
url Field that contains a url address
